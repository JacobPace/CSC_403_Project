# CSC 403 Project - Tour De Button

## About
Tour De Button is a simple website coded in Javascript, HTML, and CSS that helps a user visit various states and destinations in the USA.

## Features
  1. An interactable map of the USA provided via SimpleMaps (https://simplemaps.com/)
  2. A button that randomly selects a state to visit
  3. Displays a popup for suggested tourism destinations in that state, including 2 restaurants and 2 recreational destinations
  4. The ability to select already visited states
  5. Highlighting of visited states to keep track of which states you have left to visit
  6. A reward popup for when all states have been visited

## Usage
Set up is simple. Download the zip file in main and extract all files. Select the HTML file to open the website in a browser.
